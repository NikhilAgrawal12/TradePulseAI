# TradePulseAI

**TradePulseAI** is a web application that allows users to track stocks, buy and sell shares, view analytics, and monitor their portfolio. The platform integrates AI  based predictions to provide insights and support informed trading decisions. Future updates will include LLM and generative AI features for enhanced recommendations.

---

## **Application Pages**

### **Header (Common)**
- Visible on all pages.
- Includes navigation links, sign-in/sign-up options, and other common actions.

### **1. Stocks Page**
- Displays all stocks in a table format.
- Includes a search bar and actions: **Buy**, **Sell**, **Hold**.
- Highlights **top-performing stocks**, interesting segments, and relevant news.

### **2. Analytics Page**
- Detailed analytics and dashboards for selected stocks.
- Shows key metrics, historical trends, and AI-driven insights.

### **3. Portfolio Page**
- Shows user’s stock holdings with quantity, value, and performance.
- Calculates profit/loss for each stock and overall portfolio.
- **Sign-in required**.

### **4. Login and Registration Page**
- Allows users to **sign in** or **create a new account**.

### **5. Watchlist Page**
- Users can **save favorite stocks** to monitor easily.

### **6. Cart Page**
- Displays all stocks the user intends to buy.
- Allows users to review and adjust before purchase.

### **7. Payment Page**
- Secure checkout for purchasing stocks.

### **8. Orders History Page**
- Displays all previous orders with date, time, and transaction details.

### **9. Account Management Page**
- Users can update email, password, and account preferences.

### **10. Help / FAQ / About Page**
- Guides users on using the app, trading basics, and AI predictions.
